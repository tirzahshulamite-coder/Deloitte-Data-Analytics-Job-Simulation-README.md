# Deloitte Data Analytics Job Simulation

## Overview

This repository contains my completed work for the Deloitte Data Analytics Job Simulation offered through Forage. The simulation provided hands-on experience in solving real-world business problems using data analysis techniques in Microsoft Excel.

The project focused on analyzing datasets, validating information, classifying equality scores based on predefined criteria, and presenting accurate results to support business decision-making. This experience strengthened my analytical thinking, problem-solving, and attention to detail while demonstrating how data analytics is applied in a professional consulting environment.

---

## Project Objectives

- Analyze business data using Microsoft Excel.
- Validate and clean datasets for accuracy.
- Classify equality scores according to business rules.
- Generate meaningful insights from data.
- Present results in a clear and organized format.

---

## Tasks Completed

### Task 1 – Data Analysis
- Explored and analyzed business data.
- Identified patterns and inconsistencies.
- Prepared data for analysis.

### Task 2 – Equality Table
- Classified Equality Scores into:
  - **Fair:** -10 to 10
  - **Unfair:** Less than -10 or greater than 10 up to ±20
  - **Highly Discriminative:** Less than -20 or greater than 20
- Verified the accuracy of all classifications.
- Prepared the completed workbook for submission.
